# Step 1: Select Patient

After clicking `Import Device Exam` in the Main Menu, select an existing patient.

:::tip Best practices to avoid duplicates

- Search by last name first.
  :::

![Step 1: Select Patient](/img/import-device-exam/impDevExa-selPat_en.png)

Once the patient is selected, a new screen appears where you can attach files.
