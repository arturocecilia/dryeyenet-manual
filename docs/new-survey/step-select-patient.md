# Step 1: Select patient

Select an existing patient or add a new patient.

:::tip Best practices to avoid duplicates

- Search by last name first
- Use a consistent local Patient ID if your clinic has one
- Verify email spelling before saving (affects Patient Portal invitations)
  :::

![Step 0 Select Patient](/img/new-base-survey/newBasSur_selPat_en.png)

Once selected the patient The survey can be completed in one of the following ways:

![Step 0 Actions with selected patient](/img/new-base-survey/newBasSurActions_en.png)

1. Fill it out now within the application
2. Send to a tablet for completion by the patient on-site
3. Send to the Patient Portal, where it will appear once the patient logs in

## Substep: Add patient and Edit patient

The Add/Edit patient dialog is used both to create new patients and to correct/complete demographic data.

![Step 0 SubStep Create patient](/img/new-base-survey/newBasSur_addPat_en.png)

**Mandatory field:**

- Last name

**Strongly recommended:**

- First name
- Email address (required for Patient Portal invitations)
- Local patient identifier (if used)

## Consent

Depending on configuration, consent can be captured digitally (signature) or documented as obtained externally.

![Step 0 SubStep Create patient - Patient Consent](/img/new-base-survey/newPat_consent_en.png)
