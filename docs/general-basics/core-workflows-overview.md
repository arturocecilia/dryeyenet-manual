# Core Workflows Overview

**Typical operational sequence:**

1. Patient management
2. Perform a [**New Base Survey**](/docs/new-base-survey/intro)
3. Perform a [**New base Exam**](/docs/new-base-exam/intro)
4. Create or request a [**New Treatment Plan**](/docs/new-treatment-plan/intro) (role-based)
5. Document a [**New Treatment**](/docs/new-treatment/intro) (role-based)
6. Use the [**Patient Dashboard**](/docs/patient-dashboard/intro) to review and edit data

_In many clinics, questionnaires (especially OSDI) are completed by the patient via the Patient Portal prior to the visit._
