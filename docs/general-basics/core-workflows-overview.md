# 1.2 Core Workflows Overview

**Typical operational sequence:**

1. Identify or create the patient
2. Perform New base survey
3. Perform New base Exam
4. Import or request connected device and/or additional documents
5. Create or request a treatment plan (role-based)
6. Document treatment
7. Use Dashboard to review/edit data

_In many clinics, questionnaires (especially OSDI) are completed by the patient via the Patient Portal before the visit._
