# DryEyeNet OPHTHA

**DryEyeNet OPHTHA** is the preset designed for ophthalmologists responsible for most clinical workflows within the patient journey.

By default, this preset includes access to the following tasks:

| Task                                                 | Presence |
| ---------------------------------------------------- | -------- |
| [New Base Survey](/docs/new-base-survey/intro)       | Always   |
| [New Base Exam](/docs/new-base-exam/intro)           | Always   |
| [Import Device Exam](/docs/import-device-exam/intro) | Optional |
| [New Treatment Plan](/docs/new-treatment-plan/intro) | Optional |
| [New Treatment](/docs/new-treatment/intro)           | Always   |
| [Patient Dashboard](/docs/patient-dashboard/intro)   | Always   |
| Order Dashboard                                      | Always   |
| Automatic Device Exam Dashboard                      | Optional |
