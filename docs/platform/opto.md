# DryEyeNet OPTO

DryEyeNet OPTO is designed to be used by Pre-Treatment Centers. However, depending on the specific workflows and regulatory requirements of each country, the set of accessible tasks can be extended to include additional steps required along the dry eye care pathway.

By default, it includes the following tasks:

- **New Base Survey**
- **New Base Exam**
- **Device Import**
