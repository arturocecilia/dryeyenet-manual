# Step 1: Select Patient

After clicking `New Base Survey` in the **Main Menu**, you can **Select** an existing patient or add a new one.

:::tip Best practices to avoid duplicates

- Search by last name first.
- Use a consistent local patient ID if your clinic has one.
- Verify the email spelling before saving (this affects Patient Portal invitations).
  :::

![Step 1 Select Patient](/img/new-base-survey/newBasSur_selPat_en.png)

Once the patient is selected, the survey can be completed in one of the following ways:

![Step 1 Actions with selected patient](/img/new-base-survey/newBasSurActions_en.png)

1. Fill it out immediately within the application.
2. Send it to a tablet for completion by the patient on-site.
3. Send it to the Patient Portal, where it will appear once the patient logs in.

## Substep: Add Patient and Edit Patient

The **Add/Edit Patient** dialog is used both to create new patients and to correct or complete demographic data.

![Step 1 Substep Create Patient](/img/new-base-survey/newBasSur_addPat_en.png)

**Mandatory field:**

- Last name

**Strongly recommended:**

- First name
- Email address (required for Patient Portal invitations)
- Local patient identifier (if used)
