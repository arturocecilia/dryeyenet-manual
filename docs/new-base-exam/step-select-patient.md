# Step 1: Select Patient

After clicking `New Base Exam` in the **Main Menu**, select an existing patient.

> **Tip:** Best practices to avoid duplicates
>
> - Search by last name first.

![Step 1 Select Patient](/img/new-base-exam/newBasExa_selPat_en.png)

Once the patient is selected, the Base Exam becomes available for completion.
