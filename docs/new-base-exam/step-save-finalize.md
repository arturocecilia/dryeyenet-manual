# Step 3: Save / Finalize

Once all available information in the selectable areas has been documented, you can `SAVE` and/or `FINALIZE` the Base Exam by clicking the corresponding buttons in the bottom menu.

![Substep Save / Finalize Base Exam](/img/new-base-exam/newBasExa_save_finalize_buttons_en.png)

After completing the **Base Exam** and confirming that it is ready, click **Finalize**.

![Substep Finalize Base Exam Confirmation](/img/new-base-exam/newBasExa_save_finalize_popup_en.png)
